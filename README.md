# GigaSecretary - ваш ИИ-секретарь!

---

## Описание

Этот телеграм бот помогает управлять вашим гугл-календарём с помощью обычной человеческой переписки или речи.

## Подготовка к запуску

### Вывод в голбальную сеть

Из-за ухода ngrok пришлось искать обходные пути, поэтому данный бот тестировался с выводом в сеть через xTunnel.

Алгоритм действий для подготовки к запуску проекта:
1) Скачать xTunnel
2) Зайти в папку с xTunnel
3) Открыть в ней консоль
4) Ввести `.\xTunnel 8080`
5) Если всё верно, должно вывести Connected и показать глобальный URL

### Настройка GoogleAPI

1) Открыть `https://console.cloud.google.com/apis/credentials/consent`
2) Там зарегистрировать API для OAuth 2.0
3) В настройках URL указать скопированную с xTunnel ссылку
4) Скачать client_secret.json и вставить в корень проекта
5) Взять данные SCOPES и REDIRECT_URI в переменные окружения под переменными GOOGLE_SCOPES и GOOGLE_REDIRECT_URI соответственно
6) Подождать от 5 минут до 2 часов, гугл любит перепроверять всё по 100500 раз

### Настройка языковой модели
Языковая модель может быть использована любая, в гигачате ещё нужно будет зарегистрироваться по номеру телефона (фу)

Алгоритм для GigaChat:
1) Зайти в `https://developers.sber.ru/studio/workspaces`
2) Создать проект с моделью GigaChat
3) Из настроек API забираем ClientID, SCOPE и ACCESS_TOKEN, которые также забиваем в переменные окружения GIGA_CLIENT_ID, GIGA_SCOPE, GIGA_AUTHRIZATION_KEY соответственно

# Настройка бота
1) Создать бота в телеграмме
2) Указать в переменной окружения BOT_TOKEN токен твоего бота

## Запуск
Всё предельно просто

1) Запустить main
2) Зайти в бота
3) Пользовать в своё удовольствие
